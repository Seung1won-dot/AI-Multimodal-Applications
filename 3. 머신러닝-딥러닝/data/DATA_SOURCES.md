# 데이터 출처 (Data Sources)

이 폴더의 CSV 파일들은 용량 제한으로 GitHub에 직접 업로드하지 않습니다.
아래 링크에서 다운로드하세요.

---

## 1. 서울 자전거 공유 수요 데이터
- **파일**: `SeoulBikeData.csv`
- **출처**: https://www.kaggle.com/datasets/joebeachcapital/seoul-bike-sharing/data
- **설명**: 서울시 공공자전거(따릉이) 대여 수요 예측 데이터셋

---

## 2. 호텔 예약 수요 데이터
- **파일**: `hotel_bookings.csv`
- **출처**: https://www.kaggle.com/datasets/jessemostipak/hotel-booking-demand/data
- **설명**: 호텔 예약 취소 및 수요 패턴 분석 데이터셋

---

## 3. 주택 임대료 예측 데이터
- **파일**: `House_Rent_Dataset.csv`
- **출처**: https://www.kaggle.com/datasets/iamsouravbanerjee/house-rent-prediction-dataset
- **설명**: 주택 임대료 예측을 위한 데이터셋

---

## 4. 손글씨 도형 이미지 (shape.zip)
- **파일**: `shape.zip`
- **출처**: 해당 데이터셋 사용 시 커스텀 수집 또는 https://www.kaggle.com 에서 검색
