# 📊 데이터 분석 (Data Analysis)

파이썬을 활용한 데이터 분석 기법을 학습하는 과정입니다.

## 📚 학습 내용

### 데이터 분석 기초
- **NumPy**: 수치 계산 및 배열 처리
- **Pandas**: 데이터프레임 기반의 데이터 분석
- **Matplotlib/Seaborn**: 데이터 시각화

### 실전 데이터 분석
- **Online Retail 데이터**: 온라인 소매 데이터 EDA
- **소상공인 상가 정보**: 서울시 상가업소 분석
- **서울시 공공자전거**: 따릉이 대여 데이터 분석
- **커피 프랜차이즈**: 스타벅스 입점 전략 분석

### 웹 스크래핑
- Selenium을 활용한 웹 자동화
