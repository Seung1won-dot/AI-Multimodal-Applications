# 🐍 파이썬 기초 (Python Basics)

파이썬 프로그래밍의 기본 개념과 문법을 학습하는 과정입니다.

## 📚 학습 내용

### 기본 문법
- `print` 함수와 변수
- 파이썬의 변수 타입
- 문자열 다루기

### 컬렉션 타입 (Collections)
- **리스트(List)**: 순서가 있는 가변 시퀀스
- **튜플(Tuple)**: 순서가 있는 불변 시퀀스
- **딕셔너리(Dictionary)**: 키-값 쌍의 컬렉션
- **세트(Set)**: 중복 없는 유일한 값의 집합

### 연산자
- 산술, 비교, 논리 연산자

### 제어문
- 조건문 (if-elif-else)
- 반복문 (for, while)

### 함수
- 사용자 정의 함수
- 변수의 범위 (Scope)
- 콜백 함수와 람다 함수

### 객체지향 프로그래밍
- 클래스와 객체
- 상속 (Inheritance)
- 특수(매직) 메서드

### 예외 처리
- try-except문을 활용한 오류 처리
