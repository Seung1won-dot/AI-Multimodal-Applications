# 🤖 머신러닝 & 딥러닝 (ML/DL)

기계학습과 딥러닝의 기초부터 실전 구현까지 학습하는 과정입니다.

## 📚 학습 내용

### 머신러닝 (Scikit-Learn)
- 지도학습, 비지도학습 기초
- 분류, 회귀 모델
- 모델 평가 및 최적화

### 딥러닝 기초
- **TensorFlow**: Google의 딥러닝 프레임워크
- **PyTorch**: Facebook의 딥러닝 프레임워크
- 선형 회귀, 로지스틱 회귀 구현

### 딥러닝 핵심
- **퍼셉트론과 다층 퍼셉트론(MLP)**
- **CNN (Convolutional Neural Network)**: 이미지 분류
- **RNN, LSTM, GRU**: 시퀀스 데이터 처리
- **AlexNet**: CNN 발전의 시작점

### 실전 프로젝트
- 손글씨 숫자 분류 (MNIST)
- 손글씨 도형 분류
- Alien vs. Predator 이미지 분류
- 서울 자전거 수요 예측
- 호텔 예약 수요 예측
- 주택 임대료 예측
