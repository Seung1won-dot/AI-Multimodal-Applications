# 🔮 멀티모달 (Multimodal AI)

자연어처리(NLP), 컴퓨터 비전(CV), 그리고 두领域的融合인 멀티모달 AI를 학습하는 과정입니다.

## 📚 학습 내용

### 자연어 처리 (NLP)
- **텍스트 벡터화**: Bag of Words, TF-IDF
- **신경망 기반 벡터화**: Word2Vec, Embedding
- **RNN/LSTM/GRU**: 시퀀스 모델링
- **Seq2Seq**: 시퀀스 대 시퀀스 변환
- **어텐션 메커니즘**: 중요도에 가중치 부여
- **트랜스포머**: Self-Attention 기반 아키텍처
- **PLM (Pre-trained Language Model)**: BERT, GPT 등

### 컴퓨터 비전 (CV)
- **CNN**: 이미지 분류의 기본
- **ViT (Vision Transformer)**: 트랜스포머 기반 이미지 처리
- **오토인코더**: 차원 축소 및 특징 학습
- **DINO**: 자기지도학습 기반 비전 모델

### 멀티모달 모델
- **CLIP**: 이미지-텍스트 연결 모델
- **BLIP**: 비전-언어 사전학습 모델

### 차원 축소
- PCA, t-SNE 등 다양한 기법
